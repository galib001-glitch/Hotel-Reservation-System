Hotel Reservation System 
======================= 
 
Installation Instructions: 
1. Run "Hotel Reservation System-1.0.0.msi" 
2. Follow the installation wizard 
3. Launch from Start Menu or Desktop 
 
System Requirements: 
- Windows 10/11 (64-bit) 
- 2GB RAM 
- 200MB free space 
 
Features: 
- Guest Management 
- Room Management 
- Booking System 
- Payment Processing 
- Reports Generation
